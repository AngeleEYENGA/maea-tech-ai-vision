# MAEA Tech - AI Vision Assistant Pro 🧠

## 📱 Application de Vision par Ordinateur avec YOLOv8

Application professionnelle de détection d'objets avec géolocalisation temps réel, compatible mobile.

### ✨ Fonctionnalités

- 🎯 **Détection d'objets** avec YOLOv8 (80 classes)
- 📍 **Géolocalisation précise** via IP API
- 📸 **Analyse d'images** (unique ou multiple)
- 🎥 **Webcam temps réel** avec FPS
- 🎬 **Analyse vidéo** frame par frame
- 📊 **Analyse comparative** d'images
- 💾 **Sauvegarde automatique** des résultats
- 📱 **Interface responsive** mobile/tablette
- 🌍 **Multilingue** (Français, English, Español)
- ⚠️ **Système d'alertes** configurable

### 🚀 Installation locale

```bash
# Cloner le repository
git clone https://github.com/votre-username/maea-tech-vision.git
cd maea-tech-vision

# Installer les dépendances
pip install -r requirements.txt

# Lancer l'application
streamlit run app.py
